Nitro is a library for painlessly writing scalable, fast,
and secure message-passing network applications:

Download 0.1.1
(Jun 7, 2013) Docs
  • Millions of messages per second on commodity hardware
  • Built-in NaCL-based encryption, including certificate validation
  • Scalable‐all connection-related algorithms are O(1)
  • Fully thread-safe sockets for trivial multicore services
  • Route tracking supports proxying and RPC-style messaging
  • Pure C for easy bindings to your language
  • Runs on Linux; OS X, iOS, and Android soon
